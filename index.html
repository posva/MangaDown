Mangadown

Allows to fastly download mangas from websites using a parsing file

Download .zip Download .tar.gz View on GitHub

General Information.

Mangadown allows you to fastly download chapters from a manga. Its purpose is to support a large amount of websites using a parsing file for each. I already created 3 you can create more if you want, I will add them to the repo and credit you if you send me the parser!.
MangaDown is cross platform and depends on SFML.
I don't provide Linux binaries yet, but you can compile by yourself.

Help

Here youc an find some help about how to use MangaDown. Remember that using it under Windows or under a UNIX OS makes the command ./bin/MangaDown change to bin\MangaDown.exe. If you installed MangaDown using the pkg you can also do MangaDown. Remember you need to have the folder parsers with the parsers file inside of it in the working directory.

$ make
$ ./bin/MangaDown http://www.mangareader.net/93/naruto.html 120 140
